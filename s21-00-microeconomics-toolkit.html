Chapter 17 Microeconomics Toolkit

In this chapter we present the key tools used in the microeconomics part of this textbook. This toolkit serves two main functions:

  1. Because these tools appear in multiple chapters, the toolkit serves as a reference. When using a tool in one chapter, you can refer back to the toolkit to find a more concise description of the tool as well as links to other parts of the book where the tool is used.
  2. You can use the toolkit as a study guide. Once you have worked through the material in the chapters, you can review the tools using this toolkit.
